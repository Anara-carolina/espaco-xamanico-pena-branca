# Sobre

O **Espaço Xamânico Pena Branca** é uma aplicação web desenvolvida utilizando **React + Vite**, criada para divulgar cerimônias, medicinas da floresta e eventos realizados pelo espaço.

Além da área pública para visitantes, o sistema possui um **Painel Administrativo** protegido por autenticação, permitindo o gerenciamento das informações do site de forma simples e segura.

---

# Funcionalidades

## Área Pública

- Página Inicial
- Sobre o Espaço
- Cerimônias
- Medicinas da Floresta
- Agenda de Eventos
- Galeria de Fotos
- Formulário de Contato
- Formulário de Anamnese
- Layout Responsivo (Mobile First)

---

## Painel Administrativo

O projeto conta com uma área administrativa integrada ao **Firebase Authentication** e **Cloud Firestore**, permitindo o gerenciamento das informações sem necessidade de alterar o código da aplicação.

### Recursos disponíveis

- Login protegido por autenticação
- Cadastro de eventos
- Edição de eventos
- Exclusão de eventos
- Visualização das mensagens enviadas pelo formulário de contato
- Exclusão de mensagens
- Visualização das anamneses cadastradas
- Exclusão de anamneses
- Armazenamento em tempo real utilizando Firebase

# Instalação

Clone o repositório

```bash
git clone https://github.com/Anara-carolina/espaco-xamanico-pena-branca.git
```

Entre na pasta

```bash
cd espaco-xamanico-pena-branca
```

Instale as dependências

```bash
npm install
```

Execute o projeto

```bash
npm run dev
```
